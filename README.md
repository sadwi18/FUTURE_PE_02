## Product Overview

**Brand:** Flowlog  
**Type:** SaaS / Productivity Tool  

**One-liner:**  
The single workspace for solo founders who are done context-switching between 7 tabs.

**Core Problem Solved:**  
Freelancers and solo founders lose 2–3 hours daily jumping between Notion, Trello, calendar apps, and time trackers.  
Flowlog replaces all of them with one focused daily command center.

- Pain: Overwhelmed, scattered, losing billable hours to tool chaos

Instagram Reels · YouTube Shorts

## What Makes This Non-Generic

Most AI UGC content sounds like a press release.  
This pack was built on 4 rules:

1. **Speak first, sell second**  
   Every script opens with a real human observation, not a product claim.

2. **Specificity beats adjectives**  
   "Switching between 6 tabs every 20 minutes" > "Powerful productivity tool"

3. **Platform-native pacing**  
   - Reels: 15–30s punchy  
   - Shorts: 30–45s with slightly more story room  

4. **Hooks that earn attention**  
   Not "check out this product" — but a line that stops a mid-scroll thumb.
