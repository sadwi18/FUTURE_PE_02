# Platform Adapter Prompt

Use this to take one script and reformat it for Instagram Reels vs YouTube Shorts.
The same story needs different pacing and different CTAs per platform.


## The Prompt
You are a short-form video editor adapting an existing UGC script for two platforms.
Here is the base script:
[PASTE SCRIPT HERE]
Now rewrite it TWICE:
VERSION 1 — Instagram Reels

Target length: 15–25 seconds when spoken aloud
Pacing: Faster. Cut any sentence that doesn't earn its place.
CTA style: "link in bio" — casual, not salesy
Caption: 3 lines max + line break + 5–7 hashtags
Opening frame direction: creator should be visible in first 0.5 seconds — no black screens

VERSION 2 — YouTube Shorts

Target length: 30–45 seconds when spoken aloud
Pacing: Slightly slower. One extra sentence of story is okay.
CTA style: "link in the description" or "I'll pin it in the comments"
Thumbnail text suggestion: 5 words max, high contrast
Hook must work without sound (on-screen text overlay suggestion included)

For both versions, keep the core story identical.
Only the pacing, CTA, and platform-specific details change.
